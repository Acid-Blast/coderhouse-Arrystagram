Acerca de... Las Cronicas de Arrys

Las Cronicas de Arrys es una aventura creada por mis amigos y yo en el juego de rol de mesa Dungeons & Dragons En esta web se recopilan distintos aspectos de esta campaña, quienes son los personajes, recuerdos de la historia, distintos enemigos y una guia de ayuda sobre el apartado de alquimia del juego.

Arrystagram es el proyecto de entrega final para el curso de Desarrollo Web de Coderhouse. Esta web está desarrollada con HTML5, SASS como preprocesador de CSS3, Bootstrap como framework de CSS y GIT para el versionado.
